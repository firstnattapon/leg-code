60-second principle

ถือมูลค่าให้ใกล้ FIX_C

  1. ราคาขึ้นจนมูลค่าเกินเป้า → ขาย
  2. ราคาลงจนมูลค่าต่ำกว่าเป้า → ซื้อ
  3. ต่างจากเป้าไม่เกิน DIFF → PASS เพื่อลดต้นทุน
V = quantity × priceมูลค่าสินทรัพย์ตอนนี้
Rₙ = FIX_C × ln(Pₙ/P₀)เส้นอ้างอิงจากราคาเริ่มต้น
Eₙ = Aₙ − Rₙส่วนเกินของ price-path เหนือเส้นอ้างอิง

Reality checks

เกราะ “รู้ทัน” ก่อนเชื่อตัวเลข

เครื่องหมาย ≠ สำคัญกว่าสูตร เพราะช่วยกันนำแบบจำลองไปเรียกว่าเงินจริง

Eₙ > 0 ≠ พอร์ตกำไรต้องดู NAV, cash, position และต้นทุนร่วมกัน
Quote ≠ Fillราคาบนจออาจไม่ใช่ราคาที่ซื้อขายได้จริง
Accepted ≠ Filledส่ง order สำเร็จยังไม่ยืนยันว่าได้ quantity
Gross ≠ Netหัก fee, spread, slippage และภาษีก่อนสรุปผล
Backtest ≠ Live edgeต้องผ่านข้อมูลนอกตัวอย่างและต้นทุนจริง
HEALTHY ≠ พร้อมเทรดจริงต้องมี limits, reconciliation และ kill switch

ขอบเขตที่ต้องจำ

Aₙ/Rₙ/Eₙ ในหน้านี้เป็น educational what-if price path และยังไม่รวม fee, spread, slippage หรือภาษี ส่วน realized cash-flow ต้องมาจาก broker fill ที่มี quantity, execution price และ position reconciliation จริงเท่านั้น หน้านี้ไม่เรียก Webull API และไม่ส่ง order
